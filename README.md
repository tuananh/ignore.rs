ignore
======

Generate `.gitignore` file.

## Usage

```
ignore 0.1.0

USAGE:
    ignore [OPTIONS] --keywords <keywords>...

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -k, --keywords <keywords>...    
    -w, --write <write>             Create or update .gitignore file [default: false]
```
